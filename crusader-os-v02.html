Basic info:

Crusader OS is an hobby DIY operating system using GRUB legacy as bootloader and homemade kernel. It is written in C and assembly. It is 32-bit, runs in qemu-i386 and i haven't tested it on real hardware yet Here are some sources from i got most of the informations:

How it works

Computer → bootloader → Kernel → User

Dependencies

Debian:

sudo apt install make gcc-multilib binutils nasm grub-pc-bin xorriso qemu-system-i386 genisoimage

How to run my OS

You can either download iso of some precompiled version and run in virtual machine or compile it yourself. Compiling can be made by running command:

make a

Commands inside OS

lib - runs commands list

help - some basic info

clear - clears display

cow - makes an cow jump through your window

cat - prints cute cat

reboot - reboots using triple fault

ld - detects and checks drives

read - reads from some files

ls - reads all files from directory

dl - deletes file, use dl *file*

wr - writes into file, use wr *file* *content*

rt - runs runtest.c and make some runtests

time - shows time from RealTimeClock
